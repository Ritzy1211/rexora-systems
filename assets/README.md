# Assets

Place your images here.

## Structure

- `logos/` — Brand logos, favicons, client logos
- `images/` — Profile photos, mockups, backgrounds, and other images

## Usage in HTML

Reference files using relative paths from `index.html`:

```html
<!-- Logo -->
<img src="assets/logos/rexora-logo.svg" alt="Rexora Systems" />

<!-- Profile photo -->
<img src="assets/images/john-paulson.jpg" alt="John Paulson" />
```

## Recommended formats

- **Logos:** SVG (preferred), PNG with transparent background
- **Photos:** JPG or WebP (optimized, < 300KB)
- **Icons:** SVG
