What we'll build

1. Public school website

Modern, mobile-responsive site that converts prospective parents into admission inquiries.

  • Home, About, Academics, Admissions, News, Contact pages
  • SEO-optimized for {{LOCATION}} school searches
  • Inquiry/admission form integrated with email + WhatsApp
  • News & events publishing system

2. Parent portal

Each parent logs in to view their child's progress in one place.

  • Real-time fee balance + payment history
  • Attendance records by term
  • Termly report cards (downloadable PDF)
  • Class timetable + assignment notifications

3. Fee management system

Parents pay online. School gets notified instantly.

  • Online fee payment via card, transfer, USSD (Paystack/Flutterwave integration)
  • Automated receipts emailed to parent on payment
  • Bursar dashboard showing real-time collections
  • Automated WhatsApp/SMS reminders for outstanding balances

4. Admin dashboard

Single source of truth for school leadership.

  • Student records (enrollment, class, performance)
  • Staff records + payroll-ready exports
  • Financial reports (daily/weekly/termly)
  • User access controls (Principal, Bursar, Teachers, Parents)
