# Rexora Systems — Business Toolkit

Private workspace tools for sales, outreach, and client conversion.  
**These files are NOT deployed to the public website.** They stay local for your use.

## Files

| File | Purpose | How to use |
|---|---|---|
| `school-prospect-list.csv` | 60-row prospect tracker for cold outreach | Open in Excel/Google Sheets. Fill 3 schools/day. |
| `proposal-template.html` | Branded client proposal | Open in browser → edit `{{PLACEHOLDERS}}` → `Ctrl+P` → Save as PDF |
| `discovery-call-script.html` | 1-page sales call playbook | Print or open in 2nd monitor during calls |
| `outreach-research-guide.md` | How to find 60 schools in 3 hours | Reference before research sessions |

## Daily workflow (30 min/day)

1. **Morning (15 min):** Send 3 personalized DMs from the CSV. Update `STATUS` column.
2. **Evening (10 min):** Reply to any responses. Book discovery calls.
3. **Saturday (5 min):** Review week — replace bad leads, add 3-5 new schools.
